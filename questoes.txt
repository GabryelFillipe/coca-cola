1. O que é um repositório no Git e qual a sua importância?
R: É um espaço para guardarmos nossos projetos, sua importancia é a melhor organização e  gestão de projetos.

2. Qual a função do Git Bash?
R: Sua principal função é contraloar, criar e gerenciar projetos versionados pelo git

3. O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R: Ele seleciona o arquivo para que seja adicionado ao GitHub, o Git add <arquivo>, envia somente o arquivo selecionado, enquanto o got add. envia todos os arquivos do projeto

4. Por que utilizamos o ponto (.) no comando git add .?
R: Porque o . nos manda navegar para lugar nenhum, o que permite selecionar toods os arquivos

5. Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R: O git commit permite enviar uma descritiva do que fizemos no projeto, e é importante criar uma mensagem descritiva pois quanto trabalharmos em equipe os outros membros saberam aonde o projeto parou e o que  já foi feito.

6. Qual a função do comando git push?
R: Ele é responsavel or enviar os projetos ao GitHub, apos o git add adicionar na lista e criar o commit ele fara envio para o GitHub.

7. Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R: A principal importancia é saber as etapas do projetoque já foram feitas.

8. Explique a função das tags HTML e como elas estruturam uma página.
R: Deixar o codigos mais simples e fáceis para que outos programadores tenham um bom entendimento do codigo.

9. O que são atributos em uma tag HTML e qual a sua função?
R: São atribuidas informações adicionais que se passam para uma tag para que ela se comporte de uma forma específic

10. O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
R: É uma ferramenta que ajuda na escrida do codigo, ele gera um aestrutura básica do HTML.

11. O que é o box-sizing no CSS e por que usamos ele no reset inicial?
R: É o que define a forma e como o tamanho dos elementos são calculados, e utilizamos ele no reset inicial para definir o tamanho da "caixa" que iremos trabalhar.

12. Por que é importante utilizar tags semânticas no HTML?
R: Para facilitar a leitura do codigo com tags conhecidas, como o header, footer e etc.

13. O que são media queries no CSS e qual a sua função?
R: Sua função é ajustar a apresentação do site para diferentes dispositivos, sem modificar seu conteúdo.

14. Qual a função do GitHub Pages?
R: Sua principal função é criar dominios sedidos pelo GitHub para mantermos nossa página no ar.

15. O que é o Flexbox no CSS e para que ele é utilizado?
R: É um elemento que visa organizar os elementos de uma página de HTML dentro de seus containers de forma dinâmica.

16.Qual a diferença entre justify-content e align-items em um container Flexbox?
R: O justif-content Alinha os elementos do container na horizontal (eixo x), enquanto o align-items alinha na vertical (eixo y).

17. Com o o flex-grow funciona e qual o seu propósito no layout Flexbox?
R:  O flex-grow permite controlar como os elementos flexisiveis se expandam para preencher o espaço disponivel. Seu propósito é a criação de layouts flexisiveis, a distribuição uniforme e o preenchimento de espaço.

18. Explique a diferença entre flex-direction: column; e flex-direction: row;.
R: O flex- column organiza os itens em coluna, enquanto o flex- direction os organiza em linhas.
